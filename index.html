Checklist de Revisión de Informes

1. Verificación de Fechas

  • Revisar que las fechas mencionadas coincidan con las originales.
  • Confirmar que no haya discrepancias entre fechas de inicio, término o prórrogas.
  • Revisar la concordancia de las fechas en:
    • Texto principal
    • Anexos
    • Cuadros o tablas

2. Cuadros y Tablas de Evaluación

  • Confirmar que los datos del cuadro de evaluación sean correctos.
  • Verificar que los nombres de postores sean consistentes.
  • Revisar las fórmulas y totales.
  • Comparar con documentos de origen.

3. Datos Fundamentales del Procedimiento

  • Verificar los datos de procedencia.
  • Confirmar el número y nomenclatura del procedimiento.
  • Revisar el RUC y situación registral de las partes.
  • Confirmar número de expediente y códigos.

4. Valor Estimado / Valor Referencial

  • Verificar si el procedimiento corresponde a Valor Estimado o Valor Referencial.
  • Revisar que el Valor Estimado / Valor Referencial sea el correcto en los antecedentes.
  • Revisar que el Valor Estimado / Valor Referencial sea el correcto en la sección de procedencia.

5. Revisión Final de Ortografía y Formato

  • Revisar errores ortográficos y gramaticales.
  • Confirmar estructura lógica del informe.
  • Asegurarse de usar términos correctos.
  • Confirmar que el informe esté firmado.
